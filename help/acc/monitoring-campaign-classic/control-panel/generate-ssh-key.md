# 生成SSH密钥

您需要能够生成SSH密钥，才能将其添加到控制面板。

## 访问控制面板SFTP管理

要在控制面板中访问SFTP管理，请访问：

* [Experience Cloud主页](https://experience.adobe.com/#/home) >解决方案选取器： 活动> **控制面板** 卡 **>** SFTP卡

   或
* 直接从URL: [http://experience.adobe.com/#/controlpanel/sftp](http://experience.adobe.com/#/controlpanel/sftp)
