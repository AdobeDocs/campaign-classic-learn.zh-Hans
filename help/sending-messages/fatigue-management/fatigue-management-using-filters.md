# 使用预定义过滤器设置疲劳管理

疲劳管理控制消息传递的频率和数量，以避免收件人过度请求。 如果您的活动实例中没有活动优化模块，您可以配置一个预定义的筛选器，该筛选器将按收到的消息数来筛选目标群。
