# 监视数据库

您的活动实例的所有资源、工作流和数据都存储在数据库中。 达到最大容量的数据库可能会导致实例出现问题（无法登录、发送电子邮件等）。 因此，定期监控实例的数据库是确保最佳性能的关键。
